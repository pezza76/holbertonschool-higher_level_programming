# 1-my_list.txt

Instructions:

- Create a class MyList that inherits from list.
- Implement a method print_sorted(self) that prints the list sorted in ascending order.
- The original list should not be modified.

Example:

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]
